# breader

### Install Dependencies
```sh
$ sudo apt istall libzbar0
  or
$ sudo ... install zbar
```

```sh
$ pip install --upgrade pip
$ pip install -r requirements.txt
```

### Define Environment Variable
```sh
$ export \
  HOST=<HOST> \
  PORT=<PORT> \
  REFERRER_URL=<REFERRER_URL> \
  DEBUG=<DEBUG>

```

### Then run
```sh
$ python app.py
```
